# GrayMatter

GrayMatter is an OpenClaw AgentSkill for durable memory and graph operations through `api-0`.

Included:
- `SKILL.md`
- `scripts/graymatter_api.sh`
- packaged distributable: `graymatter.skill`

Purpose:
- read and write durable `MemoryEntry` records
- access GrayMatter graph state through production ValkyrAI endpoints
- provide a practical skill interface until MCP-native GrayMatter wiring is complete
